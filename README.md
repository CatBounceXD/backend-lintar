# Project UTS Backend Programming - Kelompo
Proyek ini adalah RESTful API yang dibangun menggunakan Node.js, Express.js, dan MongoDB (Mongoose), tugas Ujian Tengah Semester (UTS), Backend Programming.

## Code yang Digunakan
* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB Atlas (Cloud)
* **ODM:** Mongoose
* **Environment:** dotenv
